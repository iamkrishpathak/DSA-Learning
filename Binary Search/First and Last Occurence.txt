class Solution {
    public int[] searchRange(int[] nums, int target) {
        int low = 0, high = nums.length - 1;
        int firstOccurence = -1;
        int lastOccurence = -1;
        while(low<=high){
            int mid = (low+high)/2;
            if(nums[mid] == target){
                high = mid - 1;
                firstOccurence = mid;
            }
            else if(nums[mid] > target){
                high = mid - 1;
            }
            else{
                low = mid + 1;
            }
        }
        low = 0;
        high = nums.length - 1;
         while(low<=high){
            int mid = (low+high)/2;
            if(nums[mid] == target){
                low = mid + 1;
                lastOccurence = mid;
            }
            else if(nums[mid] > target){
                high = mid - 1;
            }
            else{
                low = mid + 1;
            }
        }
        int[] result = {firstOccurence, lastOccurence};
        return result;
    }
}

Time complexity - O(log n)
Space Complexity - O(1)

#APPROACH:
1) We need first and last occurence
2) Use Binary Search twice

- First Binary Search will find first occurence and Second one will find last occurence
